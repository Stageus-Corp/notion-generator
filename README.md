# 💻 **Notion API 명세서 자동 생성기**

## :key: **목표**
API 명세서를 매번 Notion에 들어가서 고치기가 힘들었다. 
폴더 하나에 명세서 관련한 내용을 JSON으로 적고 한 번에 Notion에 옮겨지도록 해보려한다.


<br/>

## 🔧 **사용 기술**
* nodejs
* Notion API

<br/>

## :open_file_folder: **폴더 구조**
```bash
├── config
│   └── setting.js
├── module
│   ├── deleteNotionData.js
│   ├── notion.js
│   ├── saveApiDoc.js
│   └── addNotionData.js
└── server.js
```
